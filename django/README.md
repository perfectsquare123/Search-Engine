# The backend of knowledge search engine

Note: Please ensure that you have completed the steps in Elasticsearch

### Start

Install django if not yet:
```bash
pip install django
```

Migrate and run:
```bash
python manage.py migrate
```
```bash
python manage.py runserver
```
